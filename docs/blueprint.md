# **App Name**: GlowHer Wellness

## Core Features:

- Cycle Tracking: A calendar view for tracking cycles with predictions and wellness tips.
- Mood Logging: Daily mood journal with interactive emojis or sliders for capturing emotional state.
- Audio Relaxation: Access to embedded YouTube Music searches or curated relaxing playlists.
- Self-Care Reminders: Daily wellness reminders and gentle tips presented to enhance self-care routines.
- AI Wellness Tool: Generative AI tool that offers custom wellness advice, factoring in user mood and cycle data. This tool provides personalized recommendations.

## Style Guidelines:

- Primary color: Muted beige (#F4E1D2) to serve as a soft, neutral base, promoting a sense of calm and well-being.
- Background color: Warm white (#FFF8F0), a lighter tint of the primary hue, creates a cozy, inviting atmosphere without stark contrasts.
- Accent color: Baby blue (#D6EAF8), analogous to the primary but distinct in its cool, airy tone, adds a refreshing contrast to buttons and interactive elements.
- Headlines: 'Playfair', a serif font, adds elegance and sophistication to headings.
- Body: 'Inter', a sans-serif font, ensures clean, readable text throughout the application.
- Modern, minimal feminine icons will be used for each section.
- Gentle hover animations and button transitions to provide a smooth user experience.
